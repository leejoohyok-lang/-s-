# 현숙's 필라테스 (Hyeonsuk's Pilates)

> **맞춤형 홈트레이닝을 위한 스마트한 필라테스 가이드 서비스**

현숙's 필라테스는 사용자의 건강 상태와 필요에 맞춰 최적의 필라테스 운동 영상을 추천하고, 운동 기록을 체계적으로 관리할 수 있도록 돕는 웹 애플리케이션입니다.

## 🌟 주요 기능 (Key Features)

### 1. 지능형 검색 및 필터링
- **자연어 검색**: "어깨가 아파요", "허리 강화 15분" 등 일상적인 단어로 검색 가능
- **태그 필터**: 부위별(목, 어깨, 골반 등), 목적별(통증 완화, 스트레칭 등), 난이도별 맞춤 필터링
- **영상 길이 필터**: 5-15분, 15-30분, 30분+ 등 시간대에 맞는 영상 선택 가능

### 2. 개인화된 영상 관리
- **저장한 영상**: 마음에 드는 운동 영상을 찜하여 나만의 리스트 구성
- **최근 본 영상**: 시청 기록을 통해 이전에 본 영상을 쉽게 확인

### 3. 운동 대시보드 및 캘린더
- **운동 기록 자동 추적**: 영상 시청 완료 시 기록이 자동으로 저장
- **캘린더 뷰**: 월별 운동 실적을 직관적으로 확인
- **누적 통계**: 총 운동 시간 및 이번 달 운동 횟수 요약 제공

### 4. 고품질 영상 플레이어
- **YouTube 연동**: 큐레이션된 최적의 필라테스 영상을 모달 창에서 즉시 재생
- **반응형 디자인**: PC와 모바일 어디서나 최적의 시청 환경 제공

## 🛠 기술 스택 (Tech Stack)

- **Frontend**: HTML5, CSS3 (Vanilla CSS), JavaScript (ES6+)
- **Storage**: Browser LocalStorage (데이터 로컬 저장 및 관리)
- **External APIs**: YouTube IFrame API (영상 재생 및 썸네일)
- **Fonts**: Google Fonts (Gowun Dodum, Outfit)

## 🚀 시작하기 (Getting Started)

본 프로젝트는 별도의 서버 설치 없이 브라우저에서 즉시 실행이 가능합니다.

1. 프로젝트 파일을 로컬 환경으로 내려받습니다.
2. `index2.html` 파일을 크롬(Chrome)이나 사파리(Safari) 등 웹 브라우저에서 실행합니다.
